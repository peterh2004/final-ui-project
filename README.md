# The Rustic Bean - Artisan Coffee House

A beautiful, responsive website for The Rustic Bean coffee shop, established in 1952.

## Features

- **Menu Pages**: Browse Coffee, Pastries, Sandwiches, and Sweets
- **Shopping Cart**: Add items to cart with quantity controls
- **Checkout**: View cart, adjust quantities, and place orders
- **About Page**: Learn about the shop's history since 1952
- **Reviews Page**: Read customer testimonials

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Coffee menu (home page) |
| `pastries.html` | Pastries menu |
| `sandwiches.html` | Sandwiches menu |
| `sweets.html` | Sweets menu |
| `checkout.html` | Shopping cart & checkout |
| `about.html` | About Us & company history |
| `reviews.html` | Customer reviews |

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Inter)

## Getting Started

Simply open `index.html` in your browser to view the website.

## Design

Based on Figma design with warm, rustic coffee shop aesthetic:

- Background: `#FFF8E7` (Cream)
